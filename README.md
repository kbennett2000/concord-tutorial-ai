# concord-tutorial-ai

> 🚧 **Under construction — T0 (front door) in progress.** During the build,
> everything merged here is a coherent, followable course up to the last
> finished lesson.

You've seen an AI invent a Bible verse. In five short lessons you'll catch it
doing that with your own code — and then fix it, yourself, so it can't.

This is the third course in the Concord learning ladder, after
[concord-tutorial-web](https://github.com/kbennett2000/concord-tutorial-web)
(your first app) and
[concord-tutorial-react](https://github.com/kbennett2000/concord-tutorial-react)
(reading real code). The full course design lives in
[`docs/v1/SPEC.md`](docs/v1/SPEC.md).
